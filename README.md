# URL shortener

Greetings reviewers!

## Installation and Usage

- clone the repo and `cd` into the folder
- to start the server:
  - `cd` into the backend folder
  - run `pipenv install` and `pipenv install --dev`
  - run `pipenv shell`
  - run `pipenv run dev`
- to start the client side:
  - `cd` into the client folder
  - run `npm i`
  - run `npm run dev`
